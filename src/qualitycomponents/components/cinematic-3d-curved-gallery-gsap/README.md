# Cinematic 3D Curved Gallery | GSAP

A Pen created on CodePen.

Original URL: [https://codepen.io/editor/wvprnwyi-the-solid/pen/019d0c8d-e9d4-74d4-95ba-356d7f0919b5](https://codepen.io/editor/wvprnwyi-the-solid/pen/019d0c8d-e9d4-74d4-95ba-356d7f0919b5).

An immersive, edge-to-edge 3D image carousel inspired by high-end agency portfolios. Built with pure semantic HTML, CSS, and GSAP. Instead of complex CSS calc() operations, this gallery uses GSAP to dynamically calculate the trigonometry of a 12-sided polygon, pushing the transformOrigin back along the Z-axis to create a mathematically perfect cylinder. Features a smooth CSS lateral mask fade and a buttery 60fps infinite rotation.
